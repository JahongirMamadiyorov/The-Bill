# Start Commands

## Terminal 1 - Backend
```
cd ~/The\ Bill/restaurant-app/backend && npm start
```

## Terminal 2 - Website
```
cd ~/The\ Bill/website && npm run dev
```

## Terminal 3 - App Metro
```
cd ~/The\ Bill/RestaurantApp && npx react-native start
```

## Terminal 4 - App Android
```
cd ~/The\ Bill/RestaurantApp && npx react-native run-android
```
